# ====================
# Running the Application
# ====================
1. Open a terminal in the project root directory
2. Start the server:
	node server.js
3. Open your browser and navigate to:
   http://localhost:3000




# ====================
# Application Pages
# ====================
Public Area:
- Landing Page:        http://localhost:3000/
- Sign Up:             http://localhost:3000/signup
- Log In:              http://localhost:3000/login

Authenticated Area:
- Dashboard:           http://localhost:3000/dashboard
- Log Out:             http://localhost:3000/logout

API Endpoints:
- Health Check:        http://localhost:3000/api/health
- Login History:       http://localhost:3000/logins




# ====================
# Running Tests
# ====================
Jasmine (Unit Tests):
  npm test
  npx jasmine spec/loginSpec.js

Cucumber.js (Acceptance Tests):
  npm run test:cucumber
  npx cucumber-js features/authentication.feature

Puppeteer (Browser Automation integrated with Cucumber.js):
  npm run test:cucumber
  npm run test:all




# ====================
# Database Management
# ====================
Clear all data (delete SQLite database):
  del /f login.db
